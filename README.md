twitter-sentiment-analysis
==========================

An approach to rate tweets based on happiness level

Built in PHP + Python
Uses <a href="https://github.com/J7mbo/twitter-api-php">twitter-api-php</a>

For rating, it uses ANEW list and FINN's list
